# Laravel Client Project Management System

A recruiter-ready Laravel portfolio project for managing clients, projects, tasks, files, deadlines, and project status from a centralized dashboard.

## Project Goal

This project is designed to demonstrate practical Laravel development skills for real business workflows, including authentication, CRUD operations, MySQL database design, dashboards, file uploads, and role-based access.

## Tech Stack

- Laravel 10/11
- PHP 8+
- MySQL
- Blade / Bootstrap
- React optional in phase 2
- Git + GitHub

## Planned Features

- Admin login and authentication
- Client management
- Project management
- Task assignment
- Team/user management
- File uploads
- Project status tracking
- Priority and deadline tracking
- Dashboard summary cards
- Search and filter functionality
- Email notification plan
- Role-based access plan
