# Features

## 1. Authentication

- Admin login
- User login
- Secure password hashing
- Logout
- Protected dashboard routes

## 2. Client Management

- Add new clients
- Edit client details
- Delete clients
- View client profile
- Search clients by name, email, or company

## 3. Project Management

- Create projects for clients
- Set project title, description, deadline, budget, and status
- Assign projects to team members
- Track project progress
- Filter projects by status and priority

## 4. Task Management

- Create tasks inside projects
- Assign tasks to users
- Set priority: Low, Medium, High, Urgent
- Set due dates
- Track task status: Pending, In Progress, Completed

## 5. File Management

- Upload project documents
- Store file name, path, type, and uploader
- Link files to projects

## 6. Comments and Notes

- Add comments on projects
- Add internal notes
- Track communication history

## 7. Dashboard

- Total clients
- Total projects
- Pending tasks
- Completed tasks
- Upcoming deadlines
- Project status summary

## 8. Activity Logs

- Track important actions
- Example: project created, task completed, file uploaded

## 9. Role-Based Access Plan

- Admin: full access
- Manager: project and task access
- Team Member: assigned tasks only

## 10. Future Enhancements

- Email notifications
- REST API support
- React frontend
- Export reports to PDF/Excel
- Calendar view for deadlines
